## プロジェクト一覧

### 1. VBA 電卓アプリ
- **概要**
#### Excel VBAで作成したカスタム電卓。
- 　AIが活躍する昨今、業務の自動化などのマクロはAIの方がよく書けます。
- 　人に要求されるのは、例示的に書かれたコードを実際の業務に用いるために正しく加工・修正することです。
- 　本アプリでは、基本的なVBAの仕様を理解した上で実際に使用が可能な状態まで１から手で書き上げ自分で確認や動作テストを行った成果を示しています。AIは使用していません。
- **技術スタック**：
  - VBA (Excelマクロ)
  - Excel関数・フォームコントロール
- **ポイント**：
  - 計算処理をボタン操作で簡単に実行可能
  - オリジナルでカスタムなUIの作成例
- **学んだこと**：
  - VBA仕様とアウトプットのポイント
  - ユーザーフレンドリーなUI設計
- **ファイル構成**：
  - `Calculator.xlsm` ：メイン電卓
  - `README.md` ：説明

### 2. Zabbix監視システム構築レポート
- **概要**
　- 仮想環境上でZabbixを用いたサーバ監視の構築手順をまとめたレポート
　- Oracle VirtualBoxを用いてUbuntuを構築し、Webブラウザから自宅PCの状態を監視
- **技術スタック**：
  - Zabbix (サーバ・エージェント)
  - Linux (Ubuntu VM)
  - Apache, MySQL
  - シェルスクリプト
- **ポイント**：
  - サーバ監視の基本設定とアラート条件の定義
  - 仮想マシンでの環境構築・IP設定・サービス起動の手順を整理
  - 課題や改善ポイントも明記（監視の弱い部分、アラート改善提案など）
- **学んだこと**：
  - インフラ構築の手順書作成
  - 運用・監視の観点からのシステム改善提案
- **ファイル構成**：
  - `Zabbix_Setup_Report.pdf` ：構築レポート
  - `README.md` ：説明
